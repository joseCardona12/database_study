5. Diferencia entre clave __Primaria__ y __foránea__
La clave primaria en una entidad como atributo es única e irrepetible y es el identificador de 
cada registro en la entidad. Por otra parte, La clave foránea es un atributo que hace referencia a la clave primaria de otra entidad. Esta llave sirve para generar ese enlace y/o relación entre las dos entidades, determinando la importancia de su valor.
Ejemplo 
 
- clients -   - products -  
id            id
name          name
cellphone     cellphone
              client_id

14. Un digrama (ER), es una representación gráfica de la base de datos. Es el primer paso para iniciar la 
creacíón de una base de datos óptima y simplificada al seguir los requerimientos correspondientes y las 
limitaciones según el contexto.
La importancia se centra en la necesidad de simplificar las relaciones y el proceso de cada entidad. Asimismo,
en comprender la base de datos. 
Al crear nuestra base, partimos de ella y no necesitamos buscar o integrar algo nuevo
cuando se establceció las limitaciones y los requimeintos desde el primer momento. Además, es un eje relevante al 
anexar un nuevo colaborador en nuestro equipo.
